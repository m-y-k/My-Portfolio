# Mohammad Yusuf Khan — Portfolio

Personal portfolio website. Static, single-file, no build step.

## Run locally

Open `index.html` in any browser. Or serve it:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Edit content

Open `index.html` in any text editor. All copy, styles, and scripts are in one file. Find the section you want to change and edit the text directly.

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. Go to **Settings > Pages**.
3. Source: deploy from branch `main`, root (`/`).
4. Your site will be live in 1-2 minutes at `https://<username>.github.io/<repo>/`.

## Deploy to Cloudflare Pages / Netlify

Drag and drop this folder onto their dashboard. Done.
